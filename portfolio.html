AntHelper

Built using Twilio's messaging API in Python for HackUCI 2019. AntHelper is a tool designed to aid freshmen and new transfer students by offering community sourced responses to commonly asked questions.

Benkyou

A website created to help students studying introductory Japanese further develop their reading comprehension and speaking skills. Users may click to generate random questions that they can answer and use for practice. Try it out here!

CleanTweets

Developed at ZotHacks 2018, CleanTweets is a tool that helps users clean up their Twitter and show a more presentable social media profile. Built using Python and Flask, it won the Hacker's Choice award.
